Welcome to HEonGPU's documentation!
===================================

**HEonGPU** is a high-performance, open-source C++ library designed to accelerate Fully Homomorphic Encryption (FHE) operations by leveraging the immense parallel processing capabilities of modern Graphics Processing Units (GPUs).

This documentation provides a comprehensive guide to the library, from installation and basic usage to deep technical details and API references.
